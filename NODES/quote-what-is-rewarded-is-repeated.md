# What is Rewarded is Repeated

## Core idea
James Clear's cardinal rule of behavior change states: 'What is rewarded is repeated. What is punished is avoided.' This governs the Solution Phase of the habit loop.

## Why it matters / connection
We learn what to do in the future based on immediate feedback. If an action feels good immediately, the brain flags it as valuable. Conversely, if it feels bad, we avoid it. Aligning immediate incentives with long-term goals is the key to behavior design.
