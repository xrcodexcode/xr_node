# Regret Minimization Framework

## Core idea
The Regret Minimization Framework is a cognitive tool where decisions are evaluated by projecting oneself to age 80 and choosing the path that minimizes the total number of long-term regrets in life.

## Why it matters / connection
This framework shifts focus away from short-term fear or greed toward long-term alignment. While famously used by Jeff Bezos to start Amazon, Buffett uses a similar long-term view, prioritizing reputation, ethical behavior, and relationship building over immediate financial payouts.
