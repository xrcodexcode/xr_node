# Local File System Access Advantage

## Core idea
The main advantage of CLI-based coding agents (like Claude Code or Codex) over web-based chat interfaces is their ability to read, analyze, and modify files directly in the local file system. This allows them to process large local datasets (e.g. hundreds of PDFs or a whole codebase) that would exceed typical web drag-and-drop limits.

## Why it matters / connection
Web interfaces force users to copy-paste or upload files manually, which creates huge friction and limits context. Local CLI agents can run search tools, compile code, read folders recursively, and run diagnostics locally. This native integration enables the agent to discover files and understand workspace architecture on its own, providing a massive efficiency boost for project execution.
