# PKM Development Phases

## Core idea
The progression of AI-assisted software development and knowledge management evolves through three main engineering phases: **Context Engineering** (writing static prompts), **Harness Engineering** (configuring local files, tools, and environments for agents), and **Loop Engineering** (running unattended loops where agents orchestrate other agents).

## Why it matters / connection
This evolutionary pipeline shows how developer interaction with AI becomes increasingly abstracted. Initially, humans focus on the prompt itself (context). Then, they construct the environment and rules for the agent (harness, e.g. using `agents.md` or `claude.md` routing files). Finally, they build systems where AI agents call other agents and check their own work autonomously (loops). Understanding these phases is crucial for scaling AI productivity.
