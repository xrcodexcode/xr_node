# Quality Checklist

Every atomic note in `02_NODES/` must satisfy the following quality checklist. Notes failing any check must be marked `#draft` and resolved.

## Quality Checklist

- [ ] **One Concept Only**: The note contains exactly one atomic concept.
- [ ] **MOC Association**: Belongs to at least one Map of Content (MOC) listed in `03_MOC/`.
- [ ] **Connected**: Has at least one incoming link (backlink) from another node.
- [ ] **Sourced**: Contains a valid reference to its source note in frontmatter and links.
- [ ] **Valid Tags**: Contains only tags registered in `tag-schema.md`.
- [ ] **No Broken Links**: All internal wikilinks point to existing files.
- [ ] **Correct Format**: Adheres to the structure defined in `node-schema.md`.
