# Prompt: Merge Nodes

You are a knowledge consolidation specialist. Analyze the two provided notes and merge them into a single, cohesive atomic concept note.

## Merge Criteria
- Preserve all unique information (facts, detailed explanations, code blocks, or quotes).
- Eliminate redundant or repeated text.
- Consolidate the core idea into 1-3 clear sentences.
- Merge the tags and ensure they conform to `tag-schema.md`.
- Consolidate outbound links and combine backlinks.
- Add the title of the deleted note to the `aliases:` metadata in the header of the merged note.
